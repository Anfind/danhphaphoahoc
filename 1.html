INTRODUCTION


I. OBJECTIVES

          1. Knowledge
- Students can name organic and inorganic substances from chemical formulas.
- Students can write chemical formulas of inorganic and organic substances from their names.
- Students can read information from 3D models, Infrared spectroscopy (IR) and Mass spectrometry (MS) of substances.
          2. Competencies
- General competencies: autonomy and self-study, communication and cooperation, problem solving and creativity.
- Specific competencies of chemistry subject:
          + Study chemistry;
          + Learn about the natural world by chemical thinking;
          + Apply knowledge and skills.
          3. Qualities
- Laborious.
- Responsibility.
- Honest.
- Benevolence.
- Patriot.

II. DESIGN AND DESCRIPTION

          This is an application designed by creating Offline HTML pages through Visual Studio Code software and the Live Server utility. The application serves the purpose of learning and teaching Chemical Nomenclature and can be used for high school students. Some advantages of the web:
(1) The application presents inorganic and organic chemical nomenclature; names are transcribed and audio pronunciations are inserted; there are some examples and exercises to reinforce knowledge in each section;
(2) The search section contains many elements, inorganic compounds, organic compounds and some commonly substances in life; The information presented includes names, chemical formulas, 3D models of organic substances with which the user can interact with molecular rotation, Infrared spectroscopy (IR) and Mass spectrometry (MS) of the organics;
(3) Provide website links to draw 2D structures, 3D structures, find information about substances;
(4) There is a game section to help users evaluate learning efficiency;
          This is a useful application for teachers and students in teaching and learning Chemistry.
